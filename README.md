# source_map_transformer 

> [pub](https://pub.dartlang.org/) transformer that transforms source
> maps.

## Usage example

### `pubspec.yaml`

```yaml
name: autoprefixer_example
dependencies:
  source_map_transformer: any
transformers:
- source_map_transformer
```

## Options

### `source_map`

Perform source map transformations in release mode.

TYPE: `bool`  
DEFAULT: `false`
